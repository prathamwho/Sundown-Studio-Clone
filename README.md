- HTML5

- JavaScript (ES6)

## 📦 Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/prathamwho/Sundown-Studio-Clone.git
   ```

2. Navigate to the project directory:

```bash
cd Sundown-Studio-Clone
```
3. Open index.html in your preferred web browser.
